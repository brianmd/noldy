# Noldy
=============

node application mold  
a mold for stamping out lots of composable node applications

# Usage

    $ git clone git@github.com:brianmd/noldy.git
    $ mv noldy <your app name>
    $ cd <your app name>
    $ rm -rf .git
    $ git init
    $ vim package.json
    -- change name and description
